## Request production access via the Developer Portal[​](#request-production-access-via-the-developer-portal "Direct link to heading")

*Please Note:* Be sure that you've thoroughly [tested your integration](https://developer.doordash.com/en-US/docs/marketplace/how_to/test_integration/) and have met all [integration requirements](https://developer.doordash.com/en-US/docs/marketplace/overview/integration_requirements) before requesting production access. Your integration will need to be certified by DoorDash in some capacity before receiving approval for production access.

In your **Developer Portal** > **Get Started** page, the below items need to be checked to proceed to certification.



To request production access, you will need to create a [support](https://developer.doordash.com/en-US/docs/marketplace/how_to/request_support) ticket within Developer Portal.

1. Once you're ready to go live, within **Developer Portal** on the left sidebar, Click **Support**.
2. Select the **Integration** as Marketplace.
3. Select the **Category** as [Credentials and Authentication].
4. Enter the **Description** and include that your integration is ready to be certified.
5. Click **Submit Request**.

## Next Steps[​](#next-steps "Direct link to heading")

* After you submit your request for production access, one of our team members will reach out with next steps.
* If all requirements are met, we will grant production access.

*Please reach out to support via Developer Portal if you have any questions regarding certification.*
